# Ames Housing Price Prediction

Прогнозирование стоимости жилья на основе данных о продажах домов в Эймсе, Айова. Проект выполнен в рамках самостоятельного изучения полного цикла Data Science: от SQL-анализа до контейнеризации ML-сервиса.

## Описание проекта

На основе датасета [Ames Housing](https://www.kaggle.com/competitions/house-prices-advanced-regression-techniques) построена модель для предсказания цены дома. Проект имитирует реальную бизнес-задачу: девелоперская компания хочет понимать, какие факторы сильнее всего влияют на стоимость, и иметь инструмент для быстрой оценки объектов.

## Решенные задачи

| Уровень | Задача | Инструменты |
| :--- | :--- | :--- |
| **1. Разведка** | Среднее/медиана, гистограмма цен, выбросы | Polars, Seaborn |
| **1. Разведка** | Зависимость цены от качества (`OverallQual`) | Barplot, SQL |
| **1. Разведка** | Поиск аномалий (`GrLivArea` vs `SalePrice`) | Scatter plot, ручной отбор ID |
| **2. Аналитика** | Рейтинг районов по средней цене | SQL, `RANK() OVER` |
| **2. Аналитика** | Влияние года ремонта на цену (когорты) | Polars, `//10*10`, Lineplot |
| **2. Аналитика** | Комбинированный признак: `OverallQual` vs `Age` | Pivot, Heatmap |
| **3. ML** | Обучение модели (`RandomForestRegressor`) | Scikit-learn, OneHotEncoder |
| **3. ML** | Важность признаков | Feature Importance, Barplot |
| **3. Production** | API-сервис | FastAPI, Pydantic |
| **3. Production** | Контейнеризация | Docker |

## Ключевые результаты

| Метрика | Значение |
| :--- | :--- |
| **Модель** | RandomForestRegressor |
| **R²** | 0.882 |
| **RMSE** | \$24,853 |
| **Топ-3 признака** | `OverallQual`, `GrLivArea`, `Neighborhood_NoRidge` |

## Бизнес-выводы

- **Качество материалов (`OverallQual`)** — главный драйвер цены. Инвестиции в отделку окупаются.
- **Площадь (`GrLivArea`)** важнее количества комнат. Открытая планировка предпочтительнее.
- **Район (`Neighborhood`)** критичен: дома в `NoRidge` и `StoneBr` стоят значительно дороже аналогов.
- Модель предсказывает цену с ошибкой ~$25k, что соответствует рыночному шуму (торг, срочность, вид из окна).
